Week 5 Notes

CH10: Testing & Debugging

Exception error is a return value used by program to deal with an error.
Exception will produce a stack trace. This will help lead to the error.
Chrome inspect is a powerful tool to know when something is not working right and to fix bugs.
Alerts are used to display something to the User.
Console.log() can be used to see what values are in your variables and also what is bring returned.
Error objects are used when an exception happens.
try,catch, finally can be used for exception handling
Test Driven development uses tests throughout the code to make sure everything is working as expected.
Tests are written sometimes before the code.
