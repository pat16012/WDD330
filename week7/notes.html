CH13: AJAX

AJAX allows the JavaScript to request resources from the server for the client
fetch() method only has one argument the URL of the resource needed.
use fetch() to send a request and then get a response
redirect() can be used to redirect to another URL.
text() takes text from the response and returns a promised resolved as a string in JS.
blob() is used to read a file with raw data and return a blob object.
json() Json is the most common AJAX response. used to transform JSON data into a JS object.
