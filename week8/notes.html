Week 8 Notes

CH8: Transforms and Transitions

Transforms:
- CSS3 transforms allow you to translate, rotate, scale and/or Skew any element on the page.
- CSS3 gives you unprecedented control over many more aspects of the elements appearance compaired with
CSS2.
- change transforms origin with transform-origin:0,0;

Translation:
a. translation allows you to move the element up/down/left/right, similar to position:
relative.
b. unlike position: relative, translation can only be moved relative to its current position.

Transforms & older browsers:
older browsers require pre-fixing to make it work. Syntax is online to support this.

Scaling:
- Scaling scale(x, y), defines the scales of an element horizontally then vertically.

Rotation:
- rotate(), rotates an element around the point of origin.

Skew:
- skew(x,y), specifies a skew along the x and y axis.

Transitions:
- Transitions are used to add animation features to your elements.


CH12: Canvas, SVG, and Drag and Drop


Canvas:
- Canvas is new in HTML5's API, allowing anything to be drawn all through javascript.
- Canvas element is made with "< canvas >< /canvas >"

SVG:
- SVG stands for Scalable Vector Graphics
- SVG is a file-format that allows the vector graphic to be described with XML.
- creating SVG element < SVG> < /SVG>.

Drag & Drop:
- The Drag & Drop api allows us to specify certain elements that are draggable, then specify what happens when
the element is dragged to a specific location.
